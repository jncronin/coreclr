This repository contains the .NET CoreCLR (https://github.com/dotnet/coreclr)
and CoreFX (https://github.com/dotnet/corefx) repositories as submodules
along with tymake (https://github.com/jncronin/tymake) scripts for building
them sufficiently for use with tysila and tysos
(https://github.com/jncronin/tysos).

Building
--------

path_to_tymake coreclr.tmk



Prebuilt versions are available at
https://www.tysos.org/files/tools/coreclr-2.0.0.zip
